# onrobot_rg6

Controller for OnRobot RG6 gripper.

# Requirements

- Python 3.7.3
  - pymodbus>=2.5.3

# Installation

	$ git clone git@github.com:takuya-ki/onrobot_rg6.git; cd onrobot_rg6
	$ pip install -r requirements.txt

# Usage

	$ python src/demo.py
